## Primary Change: Task API Migration (v2.1.16+)

Claude Code v2.1.16 replaced `TodoWrite`/`TodoRead` with four new tools:

| Old Tool | New Tool(s) | Purpose |
|----------|-------------|---------|
| `TodoWrite([{task, status}])` | `TaskCreate` + `TaskUpdate` | Create and update tasks |
| `TodoRead` | `TaskList` + `TaskGet` | List and retrieve tasks |

### New Task API Details

| Tool | Purpose | Key Fields |
|------|---------|------------|
| `TaskCreate` | Create task | `subject`, `description`, `activeForm` → status: `pending` |
| `TaskUpdate` | Update task | `taskId`, `status`, `addBlocks`, `addBlockedBy`, `owner` |
| `TaskGet` | Get task by ID | Returns full details including `blocks`/`blockedBy` |
| `TaskList` | List all tasks | Summary: `id`, `subject`, `status`, `owner`, `blockedBy` |

**Status flow**: `pending` → `in_progress` → `completed`
